+ **72번문제** Insertion Sort 구현

+ **73번문제** Bubble Sort 구현  
> 그냥 인접한 두 개의 수만 정렬한다는 느낌이다. 직관적이고, 빠르지는 않지만, 구현은 쉽다. swap이 가장 쓸모있어지는 정렬인것 같다.

+ **75번문제** Merge Sort 구현
